# 🎯 火山云部署 - 三步走

## 📦 准备工作（在本地完成）

### 1. 将代码推送到Git仓库

```bash
# 在项目目录下
cd /c/Users/NINGMEI/Desktop/ai-script-studio

# 初始化Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: AI Script Studio"

# 关联远程仓库（使用GitHub/GitLab/Gitee）
git remote add origin https://github.com/your-username/ai-script-studio.git

# 推送代码
git push -u origin main
```

**如果没有Git仓库，可以使用SCP直接上传（见方法2）**

---

## 🚀 部署步骤（在火山云服务器上完成）

### 方法1：使用Git部署（推荐）

```bash
# 1. SSH连接到火山云服务器
ssh root@your-server-ip

# 2. 下载代码
cd /tmp
git clone https://github.com/your-username/ai-script-studio.git
cd ai-script-studio

# 3. 运行自动部署脚本
chmod +x deploy.sh
sudo bash deploy.sh
```

### 方法2：直接上传部署

```bash
# 1. 在本地上传代码（使用Git Bash或PowerShell）
scp -r /c/Users/NINGMEI/Desktop/ai-script-studio root@your-server-ip:/tmp/

# 2. SSH连接到服务器
ssh root@your-server-ip

# 3. 运行部署脚本
cd /tmp/ai-script-studio
chmod +x deploy.sh
sudo bash deploy.sh
```

### 部署脚本会提示输入：

1. **数据库密码**：例如 `MyStr0ng!Pass2024`
2. **OpenAI API Key**：你的API密钥（sk-开头）
3. **域名或IP**：服务器的公网IP，例如 `123.45.67.89`
4. **Git仓库地址**：如果已上传代码，直接回车跳过

### 1. 检查服务状态


## 📋 部署后配置

### 配置HTTPS（如果有域名）

```bash
sudo certbot --nginx -d yourdomain.com
```

### 设置自动备份


## 📞 遇到问题？

### 常见问题速查

**问题：服务无法启动**
```bash
# 查看错误日志
sudo journalctl -u ai-script-studio-backend -n 50
```

**问题：无法访问网站**
```bash
# 检查防火墙
sudo ufw status
sudo ufw allow 80/tcp
```

**问题：数据库连接失败**
```bash
# 测试数据库
psql -U ai_script_user -d ai_script_studio -h localhost
```
