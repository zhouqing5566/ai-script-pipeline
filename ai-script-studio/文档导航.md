## 🎯 核心功能

### 5个专业Agent

1. **Agent 1 - 总控编剧**：生成整体大纲
2. **Agent 2 - 视觉架构师**：提取设定词典
3. **Agent 3 - 场次切片大师**：生成细纲
4. **Agent 4 - 分镜执笔**：并发生成分镜
5. **Agent 5 - 场记审核**：质检与组装

### 核心特性

✅ 严格的10-15秒镜头时长控制
✅ 并发API调用保护（最多5个）
✅ 指数退避重试机制（3s→9s→27s）
✅ JSON结构化输出
✅ 人工审核节点（Human-in-the-loop）
✅ 全局设定词典常驻
✅ 异步任务追踪

## 🚀 快速命令

### 部署到火山云
```bash
# 在服务器上
git clone your-repo
cd ai-script-studio
sudo bash deploy.sh
```

### 本地开发
```bash
# 后端
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload

# 前端
cd frontend
npm install
npm start
```

### 更新部署
```bash
cd /var/www/ai-script-studio
sudo bash update.sh
```

### 备份数据
```bash
sudo bash backup.sh
```
